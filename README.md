# Revenue-Link: USSD Tax Collection System

## Overview
Revenue-Link is a cloud-based USSD solution designed to simplify tax collection for local merchants. It allows users to check their payment status and register shops using basic mobile devices without needing internet access.


## Tech Stack
* **Backend:** Node.js & Express
* **Database:** JSON-based persistent storage
* **Gateway:** Africa's Talking USSD API
* **Deployment:** Render (Cloud Hosting)
